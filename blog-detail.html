GitHub Pages搭建个人网站

2026-03-06 前端开发 阅读：0

大家好，今天给大家分享如何用 GitHub Pages 快速搭建自己的个人静态网站，全程免费且无需服务器，新手也能轻松搞定！

一、准备工作

1. 拥有一个 GitHub 账号（没有的话去 github.com 注册）

2. 基本的 HTML/CSS/JavaScript 基础（本文用最简单的代码示例）

二、创建仓库

1. 登录 GitHub 后，点击右上角的「+」号，选择「New repository」

2. 仓库名必须是：你的用户名.github.io（例如：ChangXuanxu.github.io）

3. 勾选「Add a README file」，然后点击「Create repository」

三、编写网页代码

1. 进入新建的仓库，点击「Add file」→「Create new file」

2. 文件名填写 index.html，然后粘贴你的 HTML 代码

3. 拉到页面底部，填写提交信息，点击「Commit new file」

四、访问你的网站

等待1-2分钟后，访问 你的用户名.github.io 就能看到你的个人网站了！

后续可以继续添加 CSS 美化样式、JavaScript 实现交互功能，比如我们之前做的运势抽取、随机数生成等功能，都可以整合进去。

如果需要修改内容，直接在 GitHub 仓库里编辑文件即可，提交后会自动生效，非常方便。
